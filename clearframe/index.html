What we offer

Clearframe provides two core services: building a new site from scratch, and reworking an existing site. In both cases, the deliverable is the same: a clean, readable website that does its job.

New site (from scratch)

If you don’t have a website yet, Clearframe installs a proven one-page structure and fills it with only what matters: what you do, who it’s for, proof, and contact.

Best for: local services, small businesses, new ventures, portfolio starters.

Rework (existing site)

If you already have a site, Clearframe restructures and tightens it—clearer hierarchy, less redundancy, cleaner flow, and a stronger first impression.

Best for: cluttered pages, unclear messaging, outdated layout, messy navigation.

Launch-ready setup

Static by default: fast, secure, and inexpensive to host. GitHub Pages-friendly. If needed, we can provide simple guidance to connect a domain and publish.

Best for: people who want ownership and low ongoing cost.

What “good” means here

A stranger should understand what you do within seconds. They should see proof without hunting for it. And they should know exactly how to contact you. Everything else is optional.

Process

Clearframe stays tight so delivery stays predictable. The goal is speed without sloppiness.

  1. Intake — You send: business name, what you do, service area (if local), and contact info. Links and photos if you have them.
  2. Structure — We apply the right one-page frame and organize content in the correct reading order.
  3. Build — We implement the site with clean HTML/CSS: fast, mobile-ready, and minimal.
  4. Review — One short revision pass for factual corrections and clarity improvements.
  5. Handoff — You receive the site files. You own them. Deploy to your host of choice.

Deliverables

Standard one-page site includes

  • Hero section with clear positioning
  • Services / offerings (organized for easy reading)
  • Credibility / proof (reviews, credentials, brief “why trust us”)
  • Service area / hours (if relevant)
  • Contact block (email + optional phone)
  • Optional: gallery / featured work / small FAQ

Technical basics included

  • Fast static HTML + CSS
  • Mobile responsive layout
  • SEO basics (title/description + headings)
  • Accessible structure (semantic sections)
  • GitHub Pages compatible

Domain & publishing

If you want a custom domain (yourname.com), we can handle the setup and point it at your site. The domain itself is purchased through a registrar and is typically a low annual cost. You pay that directly.

This adds a small setup step, but it reduces friction: your site is live under a clean address without you needing to learn hosting details.

Multi-page builds and advanced features can be scoped separately. Clearframe stays minimal by default.

FAQ

Do you use WordPress / Wix / Squarespace?

By default, no. Clearframe sites are static so they load fast and cost little to host. If you need a CMS, we can scope it—but we won’t force tools you don’t need.

Can you write the copy?

Yes—within scope. We can tighten what you already have, or draft clean homepage copy based on intake.

Can you use stock photos?

Yes. If you have real photos, those are best. If not, we can use stock imagery to keep the site professional.

How do revisions work?

One short revision pass is included for corrections and clarity improvements. Additional changes can be scoped.
