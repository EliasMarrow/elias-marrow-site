Process

1) Intake

You send: business name, what you do, service area (if local), and contact info. Links and assets if you have them.

2) Build

We apply the Clearframe structure and build a fast, readable page with clean sections and proper hierarchy.

3) Review

One revision pass for corrections and clarity. Then we ship the final files and publish (or hand off).

FAQ

Do you use Wix / Squarespace / WordPress?

No by default. Clearframe is static so it loads fast and costs little to host. If you need a CMS, it can be scoped separately.

Can you write the copy?

Yes. We can tighten what you already have or draft clean homepage copy from your intake.

Do you need photos?

Not required. If you have real photos, we use them. If you don’t, we keep it clean with layout, typography, and intentional design elements—no generic stock visuals by default.

How many revisions?

One revision pass for clarity/corrections is included. Additional changes can be scoped if needed.
